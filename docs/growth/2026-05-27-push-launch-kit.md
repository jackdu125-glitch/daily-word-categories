## 7-Day Push Schedule

Day 1:
- Post to one feedback-friendly community.
- Reply to every comment.
- Track clicks with the Reddit UTM link.

Day 2:
- Post the X launch copy.
- Add one screenshot or short screen recording.
- Ask for difficulty feedback.

Day 3:
- Publish one TikTok/Shorts demo.
- Pin the site URL in profile or caption.

Day 4:
- Write down the top 10 feedback points.
- Fix any obvious wording or difficulty issues.

Day 5:
- Post a second Reddit update only if there are real changes.
- Lead with "I changed X based on feedback."

Day 6:
- Prepare Product Hunt assets but do not launch unless the core game is stable.

Day 7:
- Review traffic: visits, completion rate, shares, sign-ins.
- Decide whether to double down on word-game communities or pivot the homepage toward the broader game hub.
